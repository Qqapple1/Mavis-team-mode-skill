🧠 Mavis Team Mode

Recreates the Mavis Leader + Workers + Verifier workflow as a portable Agent Skill. Install on Zcode 3.0 (or Claude Code, Codex CLI, OpenClaw, Augment, Windsurf) and get 2-2.5× speedup on complex multi-step tasks.

Quick install

# 1. One-liner
curl -fsSL https://raw.githubusercontent.com/Qqapple1/Mavis-team-mode-skill/main/scripts/install.sh | bash

# 2. Or git clone
git clone https://github.com/Qqapple1/Mavis-team-mode-skill.git ~/mavis-team-mode-skill
bash ~/mavis-team-mode-skill/scripts/install.sh

# 3. Or from Claude Code
/plugin marketplace add Qqapple1/Mavis-team-mode-skill
/plugin install mavis-team-mode@Qqapple1

# Verify
bash scripts/install.sh --doctor

The workflow

┌─────────────────────────────────────────────────┐ │ User: "用 Mavis team mode 帮我做 X" │ └─────────────────────┬───────────────────────────┘ ▼ Phase 1: SCOPE + PLAN (Leader, ~2 min) - Decompose into 1..N subtasks - Decide parallel vs serial - Pick sub-agent type per subtask │ ▼ Phase 2: DISPATCH (parallel where possible) ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐ │ Worker │ │ Worker │ │ Worker │ │ Worker │ │ Coder │ │ Tester │ │Research│ │ Doc │ └────┬───┘ └────┬───┘ └────┬───┘ └────┬───┘ ▼ ▼ ▼ ▼ Phase 3: INTEGRATE (Leader synthesizes) ▼ Phase 4: VERIFY (spawn Verifier, separate context) ▼ ┌────────┐ ┌────────┐ │ PASS │ │ FAIL │ └────┬───┘ └────┬───┘ │ ▼ │ Phase 5: ITERATE (max 3x, back to P2) ▼ Phase 6: DELIVER (summary + path to artifacts)

Features

Parallel Workers

Decompose any task into 1..N sub-tasks, dispatch concurrently via Zcode's sub-agent system.

Independent Verifier

A separate sub-agent (different conversation) reviews the result. No self-review bias.

Iterate with guardrails

Up to 3 retry cycles before delivering with explicit caveats. Bounded cost.

Progressive disclosure

Only SKILL.md loads on trigger. Agents / examples / references load on demand.

7 sub-agent roles

leader, verifier, worker-coder, worker-tester, worker-researcher, worker-doc-writer, worker-reviewer.

4 worked examples

Refactor large module, bug hunt, new feature, research-then-implement. Each is copy-pasteable.

Defense-in-depth prototype

The included Todo app is a real working web app with CORS whitelist, body cap, threading lock, and 41 e2e tests.

Multi-platform CI

Tests on Ubuntu + macOS, Python 3.8 → 3.12. Pre-merge validation in GitHub Actions.

Why this exists

Mavis's Team Mode (Leader + Workers + Verifier) is the best multi-agent workflow we've seen. But it ships only with the M3 model in MiniMax Code. This skill brings the same workflow to any Zcode 3.0 setup — including setups running on DeepSeek, Claude, GPT, GLM, Kimi, or MiniMax M3.
